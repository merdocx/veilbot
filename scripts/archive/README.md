# Архив скриптов

Эта директория содержит скрипты, которые больше не используются в текущей работе, но могут быть полезны для восстановления или исторических целей.

## create_subscriptions_for_users_without.py

**Статус:** Архив (больше не используется)

**Описание:** Скрипт для массовой миграции пользователей с отдельных ключей на подписки.

**Почему в архиве:** Все пользователи уже переведены на подписки. Для новых случаев используется механизм `migrate_user_to_subscription` в `bot/services/subscription_migration.py`.

**Когда может понадобиться:**
- Восстановление после проблем с БД
- Новые установки системы
- Массовая миграция при необходимости

**Примечание:** Скрипт был обновлен с валидацией дат истечения перед архивированием.

## Разовая аналитика и починки лимитов (перенесено из `scripts/`)

Следующие файлы не используются приложением и CI; запуск только вручную при необходимости:

| Файл | Назначение |
|------|------------|
| `analyze_unused_files.py` | Поиск потенциально неиспользуемых файлов в репозитории |
| `analyze_vip_and_referral_bonus.py` | Разбор учёта VIP и реферального бонуса |
| `analyze_subscription_purchase_flow.py` | Аудит флоу покупки/продления |
| `analyze_subscription_traffic_limit_issues.py` | Аудит лимитов трафика по подпискам |
| `find_subscriptions_9gb_not_100gb.py` | Поиск подписок с лимитом 9 ГБ вместо 100 ГБ |
| `fix_subscriptions_9gb_to_100gb.py` | Пакетное исправление таких лимитов |

Запуск из корня проекта: `python3 scripts/archive/<имя>.py` (нужен venv и `DATABASE_PATH`).
