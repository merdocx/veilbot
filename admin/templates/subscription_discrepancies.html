Подписок с расхождениями

Описание

Эта страница показывает подписки, у которых реальный срок действия отличается от расчетного.

Важно: Показываются только подписки с расхождением более 3 дней. Подписки с меньшими расхождениями считаются корректными и не отображаются.

Формула расчета:

  • Если подписка создана ДО первого платежа: created_at + сумма(duration_sec всех платежей)
  • Если подписка создана ПОСЛЕ первого платежа: первый_платеж + сумма(duration_sec всех платежей)
  • Для каждого платежа используется duration_sec тарифа этого платежа (учитываются разные тарифы)
  • + Реферальные бонусы (30 дней за каждого реферала)

Особенности:

  • Используется subscription_id для точной связи платежей с конкретными подписками
  • Учитываются разные тарифы в одной подписке (если пользователь продлевал подписку разными тарифами)
  • Показываются только расхождения более 3 дней
  • Естественно истекшие подписки (если оба срока прошли) не показываются, кроме случаев с расхождением >30 дней
