4. `handle_free_tariff_with_protocol` - обработка бесплатного тарифа
5. `handle_invite_friend` - обработка реферальной программы
6. `get_tariff_by_name_and_price` - получение тарифа
7. `change_country_for_key` - смена страны для ключа
8. `change_protocol_for_key` - смена протокола для ключа
9. `reissue_specific_key` - перевыпуск ключа
10. `delete_old_key_after_success` - удаление старого ключа

**Глобальные переменные (для обратной совместимости):**
- `bot`, `dp`, `user_states` - устанавливаются из `bot/main.py` для декораторов `@dp.message_handler`
