# Архив документов и скриптов

Эта директория содержит устаревшие или завершенные документы и скрипты, которые больше не используются в текущей работе, но сохранены для исторических целей.

## Структура

- **Документы** - завершенные отчеты, анализы и миграции
- **Скрипты** - одноразовые скрипты и выполненные миграции

## Принципы архивирования

1. **Завершенные задачи** - документы о выполненных миграциях, исправлениях
2. **Устаревшие анализы** - старые анализы, замененные новыми версиями
3. **Одноразовые скрипты** - скрипты для исправления конкретных проблем
4. **Выполненные миграции** - скрипты миграций, которые уже применены

## Когда использовать архив

- Восстановление после проблем
- Понимание истории изменений
- Справочная информация о выполненных задачах
